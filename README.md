Node Server Template

## Packages

1. Express
2. Mongo DB
3. Websockets
4. MVC Design Pattern
5. Express Sessions
6. Routing System
7. Config files
8. Body Parser
9. Bcrypt
10. All modules included

---

## Setting up the project

1. Install Node
2. Install all needed node modules
3. Install Mongo DB
4. Run Mongo DB - command: mongod
5. Run app/app.js - command: node app.js
